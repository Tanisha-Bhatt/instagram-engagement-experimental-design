# Instagram Reel Engagement Optimization using Experimental Design

## Overview
Statistical optimization project applying Design of Experiments (DOE) to analyze factors affecting Instagram Reel engagement.

## Objective
To evaluate how multiple content-related variables influence engagement performance and identify optimal combinations for better outcomes.

## Methods Used
- 2³ Factorial Design
- ANOVA
- Experimental Design Matrix
- Interaction Effect Analysis
- Statistical Interpretation

## Tools Used
- Statistical Analysis
- Experimental Design Methodology
- Data Interpretation
- Presentation Analysis

## Key Outcomes
- Evaluated main and interaction effects of content variables
- Identified statistically relevant engagement drivers
- Applied experimental optimization to a practical digital media problem
